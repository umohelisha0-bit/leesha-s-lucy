# Leesha's Lucy Frontend

React-based frontend for the AI video call application.

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Runs on `http://localhost:3000`

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```
REACT_APP_API_URL=http://localhost:8000
REACT_APP_WS_URL=ws://localhost:8000
```

## Features

- 🎥 Real-time video calls
- 💬 AI-powered chat
- 🎤 Voice input/output
- 😊 Emotion detection
- 👤 Avatar display
- 📊 Call history
- ∞ Unlimited sessions
